# Roadmap V0

## Vision courte

Créer un prototype jouable court inspiré de Shandalar : exploration sur carte, rencontres, combats de cartes abstraits, récompenses, amélioration du deck, boss final.

## Principe V0

Ne pas implémenter les règles complètes de Magic avant d'avoir une boucle complète jouable.

## Boucle cible

```txt
carte → déplacement → rencontre → combat abstrait → récompense → deck amélioré → boss → fin de run
```
